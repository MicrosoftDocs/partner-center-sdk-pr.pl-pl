# <a name="partner-center-rest-error-codes"></a>Kody błędów REST Centrum partnerskiego

**Dotyczy:**

- Centrum partnerskie
- Centrum partnerskie obsługiwane przez firmę 21Vianet
- Centrum partnerskie dla Microsoft Cloud Niemcy
- Centrum partnerskie Microsoft Cloud for US Government

Interfejsy API REST Centrum partnerskiego zwracają obiekt JSON, który zawiera kod stanu. Ten kod wskazuje, czy żądanie zakończyło się pomyślnie, czy też nie powiodło się.

## <a name="success-responses"></a>Odpowiedzi na sukcesy

Kod stanu **2xx** wskazuje, że żądanie klienta zostało pomyślnie odebrane, zrozumiane i zaakceptowane.

## <a name="http-status-codes"></a>Kody stanu HTTP

Następujące kody stanu **4xx** i **5xx** wskazują błąd:

- 400: złe żądanie
- 401: Brak autoryzacji
- 403: dostęp zabroniony
- 404: nie znaleziono
- 405: Metoda jest niedozwolona
- 406: nieakceptowalny
- 409: konflikt, kod błędu
- 412: niepowodzenie warunku wstępnego
- 429: zbyt wiele żądań
- 500: wewnętrzny błąd serwera
- 501: nie zaimplementowano
- 502: Zła brama
- 503: usługa jest niedostępna
- 504: limit czasu bramy

## <a name="error-responses"></a>Odpowiedzi na błędy

Każda odpowiedź z kodem stanu **4xx** lub **5xx** zawiera komunikat o błędzie z dodatkowymi szczegółami dotyczącymi napotkanych warunków błędu.

W poniższej tabeli i kodzie przedstawiono schemat odpowiedzi na błąd:

| Nazwa        | Typ   | Opis                                                                                                                                            |
|-------------|--------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| kod        | ciąg | Zawsze zwracana. Wskazuje rodzaj błędu, który wystąpił. Wartość inna niż null.                                                                                  |
| description (opis) | ciąg | Zawsze zwracana. Opisuje błąd szczegółowo i zawiera dodatkowe informacje o debugowaniu. Wartość inna niż null, niepusta. Maksymalna długość to 1024 znaków. |
| dane        | array  | Zwracane tylko dla niektórych typów błędów. Lista obiektów błędów.                                                                                           |
| source      | string | Zawsze zwracana. Źródło błędu.                                                                                                              |

```json
{
  "code": <string>,
  "description": <string>,
  "data": [

  ],
  "source": <string>
}
WWW-Authenticate: OAuth realm=urn:cpsvc:cpid:{some cid}
```
